# 🏏 Virat Kohli Performance Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive Power BI dashboard analyzing the performance of Virat Kohli across different cricket formats, opponents, and venues.

The dashboard focuses on extracting meaningful insights from cricket data using data visualization and business intelligence techniques.

---

## 🚀 Features
- KPI metrics: Total Runs, Strike Rate, Average, Centuries  
- Year-wise performance trend analysis  
- Opponent-wise and venue-wise insights  
- Match result analysis (Won/Lost/Draw)  
- Interactive filters (ODI, T20, Test)

---

## 🛠 Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Cleaning & Transformation  
- Data Visualization  

---

## 📈 Key Insights
- Performance varies across formats and opponents  
- Certain venues show stronger performance trends  
- Year-wise trends highlight consistency and peak performance periods
